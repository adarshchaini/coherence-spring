Oracle Coherence Spring Integration

Version: 12.0.0-SNAPSHOT built on 2013-07-15 13:50

The Oracle Coherence Spring Integration project allows a Spring BeanFactory to provide objects to a Coherence class-scheme. A class-scheme in a Coherence configuration file is the means by which objects are made avaliable to a Coherence scheme. The most common use of class-scheme is to provide a Cache Store object for reading/writing to an external data store.

Dependency Configuration

To include this library as a Maven dependency, include the following in your pom.xml:

<dependency>
    <groupId>com.oracle.coherence.spring</groupId>
    <artifactId>coherence-spring-integration</artifactId>
    <version>12.0.0-SNAPSHOT</version>
</dependency>
