Issue Tracking

The Issue Tracking system for the Coherence Spring Integration project is hosted by java.net. It can be found here: http://java.net/jira/browse/COHSPR.

Public read-only access to the system is available for project Observers. To become an observer, simply join the cohspr.java.net project. All you need is a free java.net account. All Observers automatically receive project and issue updates.

To raise, update and close issues with the Issue Tracking system you will need to be a Contributor with a java.net account.
